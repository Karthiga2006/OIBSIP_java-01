# OIBSIP_java-01
The ATM Management System is a console-based application developed using Java, designed to simulate the real-world working of an Automated Teller Machine (ATM). This project provides a practical implementation of core Java programming concepts and demonstrates how user interaction, authentication, and transaction processing can be handled efficiently in a command-line environment.
This system is ideal for academic projects, internship submissions, and beginner-level software development practice.
